综合刷题网站 - 新版说明

一、打开方式
1. 解压 zip。
2. 双击 index.html 可本地使用。
3. 要让手机和电脑访问同一个网站，把整个“刷题网站”文件夹上传到 Netlify。

二、同步码与邀请码
新版已删除原来的统一访问密码。
现在进入网站后选择：
- 云端同步模式：输入同步码 + 邀请码才能使用该同步码。
- 单机模式：不需要同步码，但也需要输入邀请码；只在当前设备使用，不能查看/发布云端题解和评论。

默认邀请码：fengxing
同一个同步码在同一台设备上只需要输入一次邀请码；单机模式同一台设备也只需要输入一次邀请码。
管理员可以进入“题库管理” → “修改邀请码”，修改全站统一邀请码。

三、管理密码
默认管理密码：fengxingadmin
进入题库管理后可以修改管理密码。管理密码会保存到云端。

四、新增功能
1. 题目支持换行显示。
2. 题干、选项、题解、评论支持简单 Markdown：
   - **加粗**
   - `代码`
   - # 标题
   - - 列表
   - 换行
3. 使用手册改成首页右上角“使用手册”按钮。
4. 专题选择放在首页右侧，按“大专题/课程 → 小专题/章节 → 更小专题/小节”显示。
5. 大专题可以添加标签，支持专题/标签搜索。
6. 用户偏好设置：
   - 选完选项后自动提交
   - 选项位置是否乱序
   - 答对后是否自动下一题
   - 单选题在前、多选题在后
7. 可以设置本轮随机抽取多少个单选、多选、填空、简答。
8. 题库管理支持添加：单选题、多选题、填空题、简答题。
9. 填空题可以设置正确答案，多个答案用 | 或 ；分隔。
10. 简答题可以设置正确答案或参考答案。

五、Word 导入格式建议
选择题：
1. 题干（ ）
A. 选项一
B. 选项二
C. 选项三
D. 选项四
答案：B

多选题：
答案：ABCD

填空题可直接导入：
1. 中国共产党思想路线的核心是（ ）。
答案：实事求是

简答题可直接导入：
1. 简述数据库系统的特点。
参考答案：数据共享性高、冗余度低、独立性高，并由 DBMS 统一管理和控制。

导入 Word/TXT 时会提示选择题型：auto=自动识别，choice=选择题，fill=填空题，short=简答题。

六、注意
邀请码和管理密码都是前端实现，适合班级/个人刷题使用，不能当成严格的服务器权限系统。
如果需要更严格的账号系统，需要接 Supabase Auth 和 Row Level Security。

七、本版速度优化说明
1. 单机模式采用“本地题库先显示、后台检查云端题库更新”，不会再卡着等 Supabase 完整返回。
2. 云端题库、标签、管理密码会先检查 updated_at，没变化就不拉完整数据。
3. 同步模式的做题进度改成延迟合并上传，避免每点一次选项就请求 Supabase。
4. 题解和评论仍然是做完当前题后按需加载，不会进首页时全量加载。

八、可选：Supabase 索引优化 SQL
如果评论、题解、同步进度越来越多，可以在 Supabase 的 SQL Editor 里运行下面 SQL，提升查询速度：

create index if not exists idx_study_progress_sync_deck
on study_progress(sync_key, deck_key);

create index if not exists idx_question_notes_sync_question
on question_notes(sync_key, question_id);

create index if not exists idx_question_comments_question_created
on question_comments(question_id, created_at desc);
