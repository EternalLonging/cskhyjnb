编译原理题库导入说明

本版本已经把《编译原理选择题题库_带答案.docx》导入到网站中，并保留了 Word 中的题目图片。

结构说明：
- questions.js：已追加 158 道“编译原理 / 选择题题库”。
- assets/compiler/：保存每道题对应的图片。
- app.js：已允许显示本地 assets 图片，并锁定这些图片题的选项不乱序，避免图片里的 A/B/C/D 与按钮错位。

使用方法：
1. 解压本 ZIP。
2. 双击 index.html 可本地测试。
3. 上传到 Netlify 时，请上传整个文件夹，不要只上传 index.html。
4. 进入网站后，在专题/课程中选择“编译原理 / 选择题题库”。

注意：这些题目主要以图片形式显示，下面的 A/B/C/D 按钮只是用来作答。
